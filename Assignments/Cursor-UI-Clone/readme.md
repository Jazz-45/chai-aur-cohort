# Cursor Dev Tool Landing Page Clone

## 📌 Project Overview

This project is a **desktop-first recreation** of the Cursor developer tool landing page, built as part of the **Web Dev Cohort 2026** assignment.

The objective was to replicate the visual structure, layout hierarchy, and styling of the original website as closely as possible using only HTML and CSS, with no animations or interactivity.

---

## 🌐 Live Deployment

**Vercel Hosted Link:**

## 🧱 Sections Recreated

The following sections from the reference Cursor website were implemented:

1. **Top Navigation Bar**
   Logo, navigation links, and primary CTAs on a dark translucent header.

2. **Hero Section**
   Headline, CTA button, and large product UI visual with layered overlay imagery.

3. **Trusted By / Company Logos**
   Grid layout of partner/company logos.

4. **Feature Sections (3 Blocks)**
   Two-column layouts with alternating text and visuals.

5. **Feature Cards Section**
   Grid-based capability cards.

6. **Testimonials**
   Quote cards with author names and roles.

7. **Use Cases / Stories**
   Visual cards with contextual text.

8. **Changelog / Updates**
   Version updates with release dates.

9. **Team / About Section**
   Large team image with supporting description and CTA.

10. **Final Call-To-Action**
    Conversion section with primary download CTA.

11. **Footer**
    Multi-column structured footer with product, company, and legal links.

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**

  * Flexbox
  * CSS Grid
* **Google Fonts — Inter**

No JavaScript, frameworks, or CSS libraries were used as per assignment constraints.
